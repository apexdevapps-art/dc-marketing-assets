# Deep Connection — Marketing Assets

Public CDN of Deep Connection app marketing assets, served via raw.githubusercontent.com.

## Layout
- `videos/card_NNN.mp4` — 79 vertical video ads (TikTok / Instagram Reels)
- `cards/card_NNN.png` — 79 quote card images (Instagram feed / Pinterest pins)

## Why public
Used by the in-house social scheduler to feed Buffer / TikTok API / Instagram Graph API / Pinterest API. Public so Buffer's bots can fetch URLs without auth.
